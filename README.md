# Blueberry Spinning Course

A self-contained 44:56 indoor-cycling instructor guide and live ride console.

## Live features

- 14 timed ride blocks with RPM, RPE, coaching cues, and Spotify links
- Automatic transitions, countdown tones, completion states, and next-section preview
- Pause, previous, next, full-screen, wake-lock, and persistent progress
- Installable Progressive Web App with offline caching
- Downloadable instructor PDF backup

## Hosting

Published with GitHub Pages from the `main` branch root.

## Class use

Build the 14-song sequence in Spotify, disable Shuffle and Automix, set Crossfade to 0–2 seconds, press Play in Spotify, then press **Start timer** in the live guide.

This is an educational planning aid and not a substitute for recognized instructor certification, CPR/AED training, participant screening, or medical advice.
